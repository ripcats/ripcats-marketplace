# Changelog

## [1.0.0] — 2026-06-06

### Первый релиз

- Получение текста, фото, аудио, документов и альбомов
- Форум-топики: каждая сессия в своём треде, возобновление через `claudec -c`
- Запросы разрешений с inline-кнопками (свернуть/развернуть/принять/отклонить)
- Авто-разрешение (`/auto`)
- Закрытие сессии (`/close`) с очисткой истории
- HTML-форматирование входящего текста
- Контекст реплая (`reply_to_message_id`)
- Action `upload_photo` / `upload_document` при отправке файлов агентом
- Локальный часовой пояс для названий топиков (`tz` в access.json)
- Однопользовательская модель доступа через `allowFrom`
